# Stock Research Dashboard

A Streamlit dashboard for analyzing individual stock performance, risk, drawdowns, benchmark comparisons, stress periods, and company context.

## Features

- Historical stock performance analysis
- Benchmark comparison
- Drawdown analysis
- Rolling return analysis
- Annual return table
- Preset historical stress periods
- Company and sector context
- Advisor-style summary
- CSV export

## Run locally

```bash
pip install -r requirements.txt
streamlit run stock_dashboard.py
